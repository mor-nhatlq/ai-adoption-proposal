Chi Tiet Tung Phase — Du An Moi vs Du An Cu
1 Brainstorm & Design
BrainstormingFramework Literacy
▼
Du an moi
  • Socratic Brainstorming Brainstorming
    AI dat cau hoi Socratic 9 buoc: context → constraints → goals → 2-3 approaches → trade-offs. Khong viet code, chi hieu bai toan.
  • Tao Design Doc GATE
    Output: design spec file. Bao gom: goals, architecture, data flow, API contracts. Human PHAI approve truoc khi code.
  • Bat dau tu zero — tu do chon architecture HUMAN
    Khong rang buoc boi code cu. AI de xuat architecture tot nhat, team chon. Co the dung greenfield-first approach.
Du an cu
  • Scan & Hieu Codebase Truoc Framework Literacy
    AI doc toan bo code, map dependencies, hieu conventions dang dung. Framework Literacy (#13) de AI hieu "luat choi" cua du an.
  • Brainstorm Trong Rang Buoc Brainstorming
    AI brainstorm nhung phai phu hop voi architecture hien tai. Cac approach deu can backward compatible. Khong de xuat "lam lai tu dau".
  • Impact Analysis trong Design Doc GATE
    Design doc PHAI co them: files bi anh huong, risk of breaking changes, rollback plan. Human approve.
2 Planning (TDD-First Tasks)
Writing PlansSubagent Driving
▼
Du an moi
  • Breakdown thanh TDD Tasks Writing Plans TDD
    Moi task 2-5 phut: viet failing test → implement → verify → commit. Exact file paths, code blocks, test commands. Khong placeholders.
  • Subagent Assignment Subagent Driving
    Phan cong: Planner tao plan → Coder implement → Tester verify → Reviewer check. Moi agent co role ro rang.
  • Plan tu design doc truc tiep
    Plan sinh tu design doc da approve. Khong can tim hieu code cu vi chua co code. Nhanh va ro rang.
Du an cu
  • Plan Phai Reference Code Cu Writing Plans
    Moi task phai ghi ro: file nao can sua, function nao bi anh huong, test nao can update. AI doc code truoc khi tao plan.
  • Test Coverage Tasks TRUOC TDD GATE
    Buoc bat buoc cho du an cu: tasks dau tien la viet test cho code chua co test. Tao safety net truoc khi thay doi bat ky code nao.
  • Incremental Plan Subagent Driving
    Plan chia nho hon binh thuong. Moi batch chi thay doi 1-2 files. Verify sau moi batch. Khong lam nhieu cung luc.
3 Execution (Coding)
Git WorktreesExecuting PlansParallel Dispatch
▼
Du an moi
  • Git Worktrees cho Parallel Work Git Worktrees Parallel Dispatch
    Nhieu agents code song song, moi agent trong 1 worktree rieng. Backend/Frontend/DB dong thoi. Khong conflict.
  • Execute Plan tung task Executing Plans TDD
    Moi task: viet failing test truoc → implement code → test pass → commit. Khong skip buoc test. Khong commit code chua test.
  • Toc do cao tu sprint 1 Parallel Dispatch
    Khong co code cu can tim hieu. AI generate code truc tiep tu spec. Parallel dispatch (#9) cho max throughput.
Du an cu
  • Worktree de Isolate Changes Git Worktrees
    KHONG code truc tiep tren main. Dung worktree de isolate. Neu fail thi discard, khong anh huong codebase chinh.
  • Bat dau tu Bug Fixes & Small Tasks Executing Plans
    Xay dung trust truoc. Cho AI lam task nho, verify dung, roi moi tang do phuc tap. Khong de AI refactor lon ngay.
  • Sequential, Khong Parallel Luc Dau GATE
    Du an cu: lam tuan tu tung task. Verify xong moi task truoc khi lam task tiep. Parallel dispatch (#9) chi dung khi da confident.
4 Testing (TDD Mandatory)
TDDSystematic Debug
▼
Du an moi
  • RED-GREEN-REFACTOR TDD TDD
    Moi feature: viet test fail truoc (RED) → code du de pass (GREEN) → refactor cho dep (REFACTOR). Khong bo qua buoc nao.
  • Edge Case Discovery
    AI generate edge cases, boundary tests, error scenarios. Coverage cao tu dau. Khong no test debt.
  • Debug = 4-Phase Root Cause Systematic Debug
    Khi test fail: investigate → pattern analysis → hypothesis → minimal fix. Khong patch symptom, fix root cause.
Du an cu
  • Viet Test Cho Code Cu TRUOC TDD GATE
    Quan trong nhat! AI generate tests cho code chua co test. Regression safety net phai co TRUOC khi thay doi bat ky dong code nao.
  • Systematic Debug cho Legacy Bugs Systematic Debug
    Bug cu: AI khong guess fix. 4-phase debug: hieu root cause → viet failing test cho bug → fix → verify. 15-30 min thay vi 2-3h random fixes.
  • Human Verify Test Logic HUMAN
    Developer review test de dam bao test dung business logic. AI co the tao tests pass nhung khong verify dung. Dac biet quan trong voi code cu.
5 Code Review
Receive ReviewRequest Review
▼
Du an moi
  • AI Request Review Request Review
    Agent tu dong request review: check plan alignment, test coverage, code quality. Severity-based: blocking vs non-blocking issues.
  • AI Receive & Fix Feedback Receive Review
    Agent doc feedback, phan loai severity, fix blocking issues truoc khi human review. Giam thoi gian review 60%+.
  • Human Final Approve HUMAN
    Senior developer review ket qua cuoi cung. AI da xu ly 80% issues, human chi can focus vao design decisions.
Du an cu
  • Review Trong Context Code Cu Request Review
    Review check them: consistency voi patterns cu, backward compatibility, no unintended side effects tren code khac.
  • Regression Risk Assessment Receive Review
    AI list tat ca files/features co the bi anh huong. Suggest test commands can chay de verify. Khong chi review code moi.
  • Domain Expert PHAI Review HUMAN GATE
    Bat buoc: Nguoi hieu business domain review. AI khong hieu het business rules cua du an cu. Khong duoc skip.
6 Verify & Finalize
VerificationBranch Finalize
▼
Du an moi
  • Verification Checklist Verification GATE
    AI verify: code match plan? tests all pass? coverage met? no TODOs left? no placeholders? Evidence-based, khong tu claim "done".
  • Branch Finalize & Merge Branch Finalize
    AI clean up branch, squash commits neu can, tao PR summary. Merge strategy: merge commit hoac rebase tuy team convention.
Du an cu
  • Extended Verification Verification GATE
    Verify them: regression tests pass? Khong break existing features? Performance khong giam? Backward compatible?
  • Gradual Merge Branch Finalize HUMAN
    Deploy tung batch nho. Monitor sau moi merge. Feature flags neu can. Khong merge tat ca AI changes cung luc.
  • Rollback Plan PHAI Co GATE
    Moi merge phai co rollback plan. Neu something breaks, revert trong 5 phut. Khong chap nhan "chac khong sao dau".
